# CI/CD 보안 체크리스트

## 1. 접근 제어
- [ ] 모든 서비스에 HTTPS 적용
- [ ] 강력한 비밀번호 정책 적용
- [ ] 2FA 활성화 (GitLab, Harbor)
- [ ] IP 화이트리스트 설정
- [ ] RBAC(Role-Based Access Control) 구현

## 2. 네트워크 보안
- [ ] 불필요한 포트 차단
- [ ] 내부 네트워크 분리
- [ ] TLS 1.2/1.3만 사용
- [ ] 주기적인 SSL 인증서 갱신

## 3. 컨테이너 보안
- [ ] 최신 보안 패치 적용
- [ ] 컨테이너 권한 최소화
- [ ] 이미지 취약점 스캔
- [ ] 민감 정보 암호화

## 4. 모니터링
- [ ] 로그 중앙화
- [ ] 보안 이벤트 알림 설정
- [ ] 리소스 사용량 모니터링
- [ ] 접근 로그 감사
